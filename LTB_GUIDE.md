# Lustige Taschenbücher (LTB) Collection Guide

## Quick Import for LTB Collection

The Lustige Taschenbücher series has over 500 issues! Here's how to quickly add them:

### Method 1: Numbered Series (Recommended)

1. Create a new collection named "Lustige Taschenbücher"
2. Click "Bulk Import" button
3. Select "Numbered Series" tab
4. Enter:
   - **Series Name**: `Lustige Taschenbücher`
   - **Prefix**: `#` (or leave empty)
   - **Start Number**: `1`
   - **End Number**: `550` (or current highest number)
5. Click "Import" - this will create all items at once!

### Method 2: CSV Import

If you have a list with specific titles, you can use CSV format:

```
1,LTB #1 - Der Superheld
2,LTB #2 - Abenteuer in Entenhausen
3,LTB #3 - ...
```

Or just names (numbers auto-detected):
```
LTB #1 - Der Superheld
LTB #2 - Abenteuer in Entenhausen
LTB #3 - ...
```

### Method 3: Manual List

Paste a list of item names, one per line:
```
LTB #1
LTB #2
LTB #3
...
```

## About Lustige Taschenbücher

- **Total Issues**: 550+ (as of 2024)
- **Publisher**: Egmont Ehapa Media
- **First Issue**: 1967
- **Frequency**: Monthly
- **Category**: Comics

## Tips

- You can always add more items later if new issues are released
- Use the progress bar to track your collection completion
- Mark items as "owned" by clicking the checkbox next to each item
